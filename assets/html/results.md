# Results

The shared task has concluded. We thank all participating teams and especially those who have created system description papers for us all to learn from!
This page captures the results for all teams, besides the final scores you can also explore the decisions of each system on individual instances.

Once available we will link all papers on this page.

## Result Exploration
- You can explore the predictions of all systems on a selection of the data [here](/browse_dataset/). You may also browse the full dataset but beware [it can slow down your browser a lot](/browse_full_dataset/).
- You can explore each system's embeddings for Track B [here](/browse_embeddings/). We use a patched version of the [tensorboard embedding projector](https://github.com/tensorflow/embedding-projector-standalone), check out their documentation [here](https://www.tensorflow.org/tensorboard/tensorboard_projector_plugin).
